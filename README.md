# 🏋️ Fitness Buddy – AI Virtual Fitness Coach

Fitness Buddy is an AI-powered fitness coaching web application that provides personalized workout plans, nutrition guidance, stretching recommendations, and motivational support using IBM watsonx.ai.

🔗 Live Demo: https://fitness-buddy-ecru.vercel.app/

---

## 🚀 Features

- 🤖 AI Fitness Coach powered by IBM watsonx.ai
- 💪 Personalized workout suggestions
- 🥗 Indian nutrition recommendations
- 🧘 Stretching and mobility guidance
- 🎯 Motivational fitness support
- 📱 Responsive and modern UI
- ☁️ Deployed on Vercel

## ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/fitness-buddy.git
cd fitness-buddy
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
IBM_API_KEY=your_api_key
IBM_PROJECT_ID=your_project_id
IBM_URL=https://eu-gb.ml.cloud.ibm.com
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:8080
```
